# Lingobase: Web Tabanlı İngilizce Öğrenme Platformu

Lingobase, yapay zekâ destekli chatbot ve oyunlaştırma unsurlarıyla kullanıcıların İngilizce öğrenme sürecini kolaylaştırmayı ve daha motive edici hâle getirmeyi amaçlayan web tabanlı bir dil eğitim platformudur.

## 🔧 Kullanılan Teknolojiler

### Backend
- Node.js
- Express.js
- Sequelize ORM
- MySQL
- JWT (kimlik doğrulama)
- OpenAI ChatGPT API

### Frontend
- Vue.js 3
- Vite
- TailwindCSS
- Axios

### Diğer Araçlar
- Postman (API testleri için)
- Trello (proje yönetimi)
- Git (versiyon kontrol)

## 🚀 Kurulum Talimatları

### Gereksinimler
- Node.js 18.x+
- npm 9.x+
- MySQL 8.x+

### Adımlar

#### 1. Veritabanı
- `lingobase` adında bir veritabanı oluştur.
- `.env` dosyasını yapılandır (örnek `.env.example` dosyasını kullanarak):
  ```env
  DB_USER=your_db_user
  DB_PASSWORD=your_password
  DB_NAME=lingobase
  DB_HOST=localhost
  OPENAI_API_KEY=your_openai_key

npm init -y
npm install express sequelize mysql2 cors dotenv jsonwebtoken bcryptjs

cd backend
npm install
npm start

cd frontend
npm install
npm run dev
